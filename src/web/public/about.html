Redis Discovery

Version 0.1.0

What this is

A cross-platform utility that safely discovers Redis-compatible databases (Redis OSS, Redis Enterprise, Valkey, KeyDB where possible) on networks you are authorized to scan, and provides read-only inventory through this Web UI and the rscan CLI. Credential Scan covers the complementary case — a known list of hosts (e.g. from a CMDB export) with a username/password per host, scanned in one pass instead of a range.

Principles

  • Read-only by default — no data-modifying commands are ever sent.
  • No credential persistence — passwords are used only for the request in flight, never stored or logged.
  • Local-first — runs entirely on your machine; results are not sent anywhere.
  • Cross-platform and protocol-aware discovery.
  • Operator-friendly — clear CLI and Web UI, both backed by the same scan engine.

Out of scope

  • No writes to any scanned instance.
  • No credential storage.
  • No brute-force or credential-guessing behavior.
  • No scheduled or recurring scans.
  • No interactive command console.

Use responsibly

Only scan networks and hosts you are authorized to test.
